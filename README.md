# MCTS-based Combat Strategy Optimization

A lightweight turn-based battle simulator and Monte Carlo Tree Search (MCTS) framework for exploring decision intelligence in game AI.

## Overview
- Simulates 5v5 turn-based combat with discrete action space.
- Implements standard MCTS with UCB1 node selection.
- Baseline comparison: random strategy vs. MCTS-guided strategy.
- Planned extension: neural value network for guided search.
